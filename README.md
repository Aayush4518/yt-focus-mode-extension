# 🎯 YouTube Focus Mode – Chrome Extension

A minimal Chrome extension that helps you stay focused while using YouTube.

This extension hides the entire YouTube homepage feed during a focus session and restores it automatically when the timer ends. It is designed to reduce distractions and encourage intentional usage.

---

## ✨ Features

- 🧘 Focus Mode for YouTube  
  Hides the complete YouTube homepage feed to prevent distractions.

- ⏳ Custom Focus Timer  
  Set a timer for your work session. The homepage is restored automatically when the timer ends.

- 🎨 Minimal UI  
  Clean and simple interface with a distraction-free design.

- ⚡ Lightweight & Fast  
  No unnecessary complexity. Works instantly after activation.

---

## 🛠️ How It Works

1. Enable Focus Mode.
2. The YouTube homepage feed is removed.
3. A countdown timer starts.
4. When the timer reaches zero, the homepage is restored automatically.

This allows you to:
- Watch specific tutorials without getting distracted
- Avoid recommendation loops
- Stay productive during study/work sessions

---

## 📦 Installation (Development Mode)

1. Clone or download this repository.
2. ```npm install``` in terminal.
3. Open Chrome and navigate to:

```chrome://extensions/```

3. Enable **Developer Mode** (top-right corner).
4. Click **Load Unpacked**.
5. Select the extension folder.

---

## 🔐 Permissions

This extension only uses the necessary permissions required to:
- Modify YouTube homepage content
- Run the focus timer

No personal data is collected, stored, or transmitted.
